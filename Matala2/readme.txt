In order to run the server:

    Run the server with the path.

In order to run the client
    Run the client with the path to the files folder for upload/download. G for download (GET), P for upload (POST)
    the requested remote path is hardcoded.

    Example:
    ./simple_client <PATH_TO_FILE/example.txt> G - download file exmaple.txt from the server
    ./simple_client <PATH_TO_FILE/example.txt> P - upload file example.txt to the server
